# Repositorio de Proyectos

Este repositorio reúne varios proyectos desarrollados en distintas etapas y para diferentes propósitos. Cada carpeta contiene un módulo, script o aplicación independiente con su propia lógica y dependencias. Este archivo sirve como índice general y guía para navegar el contenido.

- ** Si Quisieras automatizar alguno de los proyectos, puedes hacerlo descargando el script y automatizandolo en el programador de Tareas.
